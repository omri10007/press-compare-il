# press-compare

Compare how different news sources cover the same story.

## What it does

- Browse a list of news events
- Click any event to see how multiple outlets covered it
- Compare headlines, summaries, and key points side by side

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Data

Sample events live in `data/events.json`. Edit that file to add or change events — no code changes needed.

All URLs in the mock data are homepage roots (e.g. `https://www.haaretz.com`), not links to specific articles.

## Tech stack

Next.js 16 · TypeScript · Tailwind CSS v4
